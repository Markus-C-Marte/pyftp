# PyFTP - Python FTP Transfer Utility

A Python-based utility for FTP file transfers between systems.

## Overview

Simplifies FTP operations with a Python interface for transferring files between multiple systems.

## Project Structure

```
.
├── pc1/    # System 1 configuration
├── pc2/    # System 2 configuration
└── [utility scripts]
```

## Status

In development. Core transfer functionality is operational.

## Technologies

- Python 3
- FTP protocol
